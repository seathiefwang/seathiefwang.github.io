python去除空格和换行符

  • python

去除空格

使用python自带的函数 strip()

  1. 去除前后空白字符

    “ xyz “.strip() # returns “xyz”

  2. 去除前面空白字符

    “ xyz “.lstrip() # returns “xyz “

  3. 去除后面空白字符

    “ xyz “.rstrip() # returns “ xyz”

  4. 替换法去除所有空格

    “ x y z “.replace(‘ ‘, ‘’) # returns “xyz”

去除换行符

使用replace()函数进行替换

"xyz\n".replace("\n","")    #return "xyz"

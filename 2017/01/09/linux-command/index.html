Linux's commands

  • Linux
  • cmd

Command

Linux下有许多的命令，在这里我记录一些常用或者有趣的命令。

Command Platform Description
ls c/c++ 显示目录下的文件与目录列表
cd c/c++ 去往指定的工作目录
find c/c++ 查找文件的工具
grep c/c++ 在指定内容中搜索指定的字符
vi c/c++ 文本编辑工具
bc c/c++ 一个基于命令行的计算器工具
sl c/c++ 常使用的ls工具的错误版，这个命令会在屏幕中打印一个移动的小火车
supervisor python 守护进程，可以管理你的服务器进程
sslocal c/c++ 科学上网工具，可以在后台运行
proxychains c/c++ 让终端命令通过代理链接网络，可配合sslocal使用
figlet c/c++ 可生成ASCII风格标题字，适合添加到你的代码注释中
yes c/c++ 不停的打印输入的字符，适合发呆和自动化处理
cowsay c/c++ 一头会说话的奶牛，还可以设置性别
ascii c/c++ 快速查看ASCII码，像一本没啥太大作用的字典
tmux c/c++ 一个神器，可以让终端多窗口运行，SSH链接时，必用
dstat c/c++ 监控你的系统资源使用状态
touch c/c++ 创建一个空文件或者文件夹
who c/c++ 查看用户登录信息
tpp c/c++ 终端下的keynote
htop c/c++ 进程，内存，IO，CPU的监控工具
curl c/c++ http工具箱
wget c/c++ 好用的下载器
calcurse c/c++ 日历系统
